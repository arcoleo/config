h3. Cedet

If you want to use Cedet, you must check out from cvs.

<pre><code>
cvs -z3 -d:pserver:anonymous@cedet.cvs.sourceforget.net:/cvsroot/cedet co -P cedet
</code></pre>

Compile -

<pre><code>
$ cd cedet
$ make
</code></pre>

h3. Rope

<code>
$ cd /path/to/repos
$ mkdir -p agr/rope
$ cd agr/rope
$ hg clone http://bitbucket.org/agr/rope
$ hg clone http://bitbucket.org/agr/ropemacs
$ hg clone http://bitbucket.org/agr/ropemode
$ cd ropemacs
$ ln -s ../ropemode/ropemode ropemode
$ cd ../rope
$ sudo python setup.py install
$ cd ../ropemacs
$ sudo python setup.py install
</code>

Quick test : you should be able to import ropemacs in a python shell.
